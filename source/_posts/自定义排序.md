## 题型1
![](https://cdn.staticaly.com/gh/heroding77/pic_bed@main/screenshot/自定义排序1.png)
**解题思路：**
题目如果换种说法就很好理解了，其实就是田忌赛马，关键是如何实现。田忌赛马的核心就是，遇到对面的好马，选比他的好一点的进行比较，如果当前马差到对面最差的马都比不过，就和对面最强的比（直接放水），那么如何实现呢？nums2由于是目标数组不能排序，但是可以将它的下标拿出来进行数组值从小到大的自定义排序，同时nums1可以直接排序，这样对于nums1，我们从最小的数和nums2的排序后下标对应的数字进行比较，用双指针分别指向nums2最大数的下标和最小数的下标，如果nums1中的数大，那么nums2对应小的数的位置放置nums1中的数，反之，nums2中大数指针对应的位置放置nums1的数，代码如下：

```cpp
class Solution {
public:
    vector<int> advantageCount(vector<int>& nums1, vector<int>& nums2) {
        int n = nums1.size();
        vector<int> index(n);
        vector<int> ans(n);
        iota(index.begin(), index.end(), 0);
        // 对nums1排序
        sort(nums1.begin(), nums1.end());
        // 对nums2的下标排序
        sort(index.begin(), index.end(), [&](int& i, int& j) {
            return nums2[i] < nums2[j];
        });
        int left = 0, right = n - 1;
        // 田忌赛马
        for(auto& num : nums1) {
            if(num > nums2[index[left]]) {
                ans[index[left ++]] = num;
            } else ans[index[right --]] = num;
        }
        return ans;
    }
};
```
