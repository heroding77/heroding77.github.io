## 题型1【记忆化数组】
![](https://cdn.staticaly.com/gh/heroding77/pic_bed@main/screenshot/滑动窗口题型1.png)
**解题思路：**
该题面对的滑动窗口问题不同于普遍的滑动窗口题型，经典的滑动窗口，元素一般要求连续的（比如连续1的个数），或者元素的个数有限，本题中窗口内的数字标签是大规模的，但是这也不影响解题思路，首先定义好数组存储当前滑动窗口内每种花的个数，定义左右指针划定滑动窗口范围，右指针不断向右寻找，并更新count数组，直到超过了边界，这时候不断让左指针右移，缩短滑动窗口大小，并更新count数组，直到当前窗口的最多个数的花不超过边界，更新总数，代码如下：
```cpp
class Solution {
public:
    const int MOD = 1e9 + 7;
    int count[100010];
    int beautifulBouquet(vector<int>& flowers, int cnt) {
        int n = flowers.size();
        int res = 0;
        int cur = 0;
        for(int i = 0, j = 0; i < n; i ++) {
            int num = ++ count[flowers[i]];
            cur = max(cur, num);
            while(cur > cnt) {
                count[flowers[j]] --;
                if(flowers[j] == flowers[i])
                    cur --;
                j ++;
            }
            res = (res + i - j + 1) % MOD;
        }
        return res;
    }
};
```
